# 💻 담당한 기능
- 테스트
    - 4가지 질문 중 답변하기
- 결과
    - 답변을 토대로 미래의 사윗감 도출
- 결과 저장
    - png로 저장
- 카카오톡 공유
  - 카카오api로 구현
